¿QUE ES GITHUB?

Es un repositorio online gratuito que permite gestionar proyectos y controlar versiones de código

¿QUIEN CREO GITHUB?
El servicio GitHub fue desarrollado por Chris Wanstrath, P. J. Hyett, Tom Preston-Werner y Scott Chacon usando Ruby on Rails, y comenzaron en febrero de 2008.
¿EN QUE SE DIFERENCA CON GIT?
GitHub es una plataforma donde puedes crear y mantener software e interactuar con otros desarrolladores. Git es el programa que utiliza GitHub para hacer que su plataforma sea perfecta y funcional
